Featured Projects

Terraplot

  • Terraplot is a full-stack web application that allows users to designate plots of farmland for certain crops and vehicles being used on those plots
  • Plots are integrated using field boundary drawing tools with Google Maps integration, automated area calculations, and real-time satellite visualization.
  • Includes an equipment management system with fuel tracking and detailed logging.
  • Also includes a feature that accesses real-time USDA crop yield data to project revenue using current market prices and compare different crop scenarios for optimal planning
  • Login information and farm data are secured with encrypted User authentication and authorization, individual farm data isolation, data storage, and encryption using Firebase, and real-time data updates
  • Tech Stack: React.js, Google Maps API, Firebase (Auth/Firestore), Tailwind CSS
  • Data Sources: USDA Crop Database, ISRIC API

TIGER TRADES

  • Used a combination of MySQL for database management and AWS for hosting the website.
  • Used HTML, CSS, Java, and SQL for webpage format, backend functionality, and database management.
  • Tiger Trades is a full-stack website that allows students to put items up for auction and also bid for items that other students have decided to put up for auction.
  • Tiger Trades has many features, such as encrypted sign-up/log-in functionality, a wallet you can add funds to for bidding, a seller dashboard for items you have put up for auction, and a page for buyers to see what items they have bid on.

AI Loan Predictor

  • An ensemble ML model that predicts if someone will be approved for a loan or not
  • Dataset: 45,000 synthetic loan applications from Kaggle (77.5% rejected, 22.5% approved)
  • Combines five classifiers (Logistic Regression, Random Forest, XGBoost, LightGBM, Gradient Boosting) using soft voting ensemble
  • Key techniques: SMOTE-Tomek resampling for class imbalance, extensive feature engineering (risk indicators, interaction terms), and threshold optimization (0.6312 vs default 0.5)
  • Results: 92.4% accuracy, 90.1% precision, 89.5% recall, AUC of 0.9713
  • Explainability: Identifies positive factors (high credit score, low debt-to-income) and negative factors (previous defaults, short credit history) for each decision
  • Application: Streamlit web app supporting individual and batch predictions with visual explanations
